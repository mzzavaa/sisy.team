# sisy.team — Organizer & B2B Tools

**Domain:** sisy.team
**Role:** B2B SaaS — Event organizer tools
**Priority:** Phase 2
**Notion:** See SISY Domain Strategy page

## Purpose
B2B platform for event organizers and companies to discover, manage, and book speakers. Provides team collaboration tools, speaker shortlists, and event planning features.

## Key Features / Sections
- Organizer dashboard & team accounts
- Speaker search & discovery with filters
- Shortlist & comparison tools
- Booking request management
- Event planning workspace
- Invoice & contract management
- API access for integrations

## Monetization
€49-€399/year B2B SaaS subscriptions based on team size and features.
